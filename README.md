# CryptoVault-DB
Proiect SI
Sistem local de management al cheilor de criptare și analiză a performanțelor
Acest proiect implementează un sistem integrat pentru gestionarea cheilor criptografice și a fișierelor criptate/decriptate, folosind Python și o bază de date relațională (MariaDB/MySQL prin SQLAlchemy). Aplicația suportă algoritmi simetrici (ex. AES) și asimetrici (ex. RSA), utilizând executabilul OpenSSL prin subprocese. De asemenea, include un framework alternativ pentru a realiza un benchmarking comparativ al performanțelor de execuție (timp și memorie) la criptare și decriptare.
